Muse is a music server that allows you to browse and listen to your music collection from a web browser.

# Features
- Low resource usage. Runs quite well in low ram environments.
- Supports (and shows!) most metadata.
- Supports most major audio formats tagged with id3 or vorbis metadata.
  - This includes mp3, ogg, opus, and flac.
- Supports transcoding of audio with ffmpeg.
- SSO by default
- Automatic library monitoring
