<Callout type="warn" title="Important">
Currently, the usage of integrations is NOT optional.
</Callout>

# Last.fm
Muse can use last.fm to get artist biographies. You'll need to set the environment variables `FM_KEY` and `FM_SECRET`

To get those, create an API account here: https://www.last.fm/api/account/create

# Spotify
Muse can use Spotify to get artist images. You'll need to set the environment variables `SPOTIFY_ID` and `SPOTIFY_SECRET`.

To get the environment variables, create an API application here: https://developer.spotify.com/dashboard/applications
